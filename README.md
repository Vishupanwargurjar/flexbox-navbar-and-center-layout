# Flexbox Layout Practice

This project is a simple practice project created using **HTML5 and CSS3** to understand the basics of **CSS Flexbox** and layout alignment.

---

## 📌 Project Overview

The goal of this project is to practice how Flexbox works for arranging and aligning elements in modern web layouts.

This project includes a navigation bar and an example of centering a div inside another div using Flexbox properties.

---

## 🚀 Features

- Navigation bar with 4 menu options
- Equal spacing between navigation items using Flexbox
- Centering a div inside another div
- Understanding alignment properties in Flexbox

---

## 🛠 Technologies Used

- HTML5
- CSS3
- CSS Flexbox
